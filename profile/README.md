## 👋 ยินดีต้อนรับสู่ TH Open Source Community

> *"เราเชื่อในพลังของ Open Source และการแบ่งปันความรู้เพื่อสร้างอนาคตที่ดีกว่า"*

เราคือชุมชนนักพัฒนาที่มุ่งมั่นสร้าง พัฒนา และแบ่งปัน Open Source Project ด้านเทคโนโลยีเครือข่าย, IoT, Network Slicing และโครงสร้างพื้นฐานดิจิทัลยุคใหม่

## 🚪 How to Join

เราเปิดรับสมาชิกใหม่เสมอ! ทำตามขั้นตอนนี้เพื่อเข้าร่วมชุมชน 👇

**ขั้นที่ 1** — มี GitHub Account (ถ้ายังไม่มี สมัครได้ที่ [github.com](https://github.com))

**ขั้นที่ 2** — ส่งข้อมูลของคุณมาที่เรา:

| ช่องทาง | วิธี |
|--------|-----|
| 📧 **Email** | ส่ง GitHub username มาที่ `contact@example.com` |
| 💬 **Facebook** | คอมเมนต์ GitHub username ในกลุ่ม |
| 🎮 **Discord** | พิมพ์ GitHub username ใน channel `#join-request` |

**ขั้นที่ 3** — รอ Admin ส่ง Invitation ให้ทาง Email (~24 ชั่วโมง)

**ขั้นที่ 4** — กด **Accept invitation** ในอีเมล ✅ เป็นสมาชิกทันที!

> 💡 **ไม่มีค่าใช้จ่าย** — เปิดรับทุกคน ทุกระดับ ทุกสาขาอาชีพ
