## 👋 ยินดีต้อนรับสู่ Open Source Community (5G)

> *"เราเชื่อในพลังของ Open Source และการแบ่งปันความรู้เพื่อสร้างอนาคตที่ดีกว่า"*

เราคือชุมชนนักพัฒนาที่มุ่งมั่นสร้าง พัฒนา และแบ่งปัน Open Source Project ด้านเทคโนโลยี **5G, IoT, Network Slicing** และโครงสร้างพื้นฐานดิจิทัลยุคใหม่
